# Healthcare-app-with-chatbot
# 🏥 Mini Healthcare Support Web App

## 📌 Project Overview

This is a simple web-based healthcare support application designed to help patients and volunteers communicate basic health concerns. The app allows users to submit their issues and receive quick guidance through an automated chatbot and summary feature.

---

## 🚀 Features

### 📝 1. Support Request Form

* Users can submit:

  * Name
  * Email
  * Health Issue / Message
* Designed for patients or volunteers seeking help

### 🤖 2. AI-Based Text Summary

* Automatically generates a short summary of the user’s input
* Helps NGOs quickly understand and prioritize requests

### 💬 3. Healthcare Chatbot (Automation Feature)

* Rule-based chatbot for instant responses
* Answers basic queries like:

  * Fever guidance
  * COVID precautions
  * Doctor consultation
* Provides quick assistance without manual intervention

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Deployment:** Vercel

---

## 🧠 AI / Automation Idea

The application includes:

* A **text summarization feature** that converts long user messages into short summaries
* A **rule-based chatbot** that provides instant responses to common healthcare questions

These features help reduce manual effort and improve response time for NGOs.

---

## 🌍 NGO Use Case

This app can be used by NGOs or healthcare organizations to:

* Collect patient or volunteer requests
* Quickly review issues using summaries
* Provide instant responses via chatbot
* Improve efficiency in handling large numbers of requests
